Chatbot-PHP
===========
A chatbot for use with [Wikia](http://wikia.com) chat rooms.

## Requirements
* PHP 5.3+
* MySQL
* A Wikia account

## Setup
* Extract files into a directory
* Edit chatbot.ini and add your bot credentials
* Run `chatbot.php`
